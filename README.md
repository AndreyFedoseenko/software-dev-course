Материалы курса "Разработка ПО" для студентов ХНУРЭ
===================

# Мотивация

ВУЗы не в состоянии выпускать людей, готовых работать на позиции Junior Developer. Основная причина, на мой взгляд, - люди, которые профессионально занимаются разработкой, не преподают в вузах, и наоборот, те, кто преподают, не занимаются разработкой. Я хочу исправить это, для начала, в рамках факультатива для заинтересованных студентов.

Проект не коммерческий, за ним не стоят крупные (или другие) компании и является моей личной инициативой. С другой стороны, не исключено, что по мере продвижения различные компании будут выступать в качестве партнёров, экзаменаторов, работодателей для выпускнков etc

# Основные принципы

1. Чтобы хорошо делать своё дело, его нужно любить. Я не буду заставлять студентов делать ДЗ, ставить им плохие оценки или запрещать посещать лекции - каждый сам для себя решает, нравится ему или нет, и сколько времени нужно уделять этому. Единственной причиной исключения может быть намеренное и постоянное нарушение рабочего процесса.
1. Программирование - это не только написание кода. В рамках курса мы будем рассматривать различные инструменты и процессы, применяемые в современной разработки ПО.
2. Быть программистом и знать язык программирования Х - две разные вещи. В течении курса многие задания будут выполняться как минимум на 2х языках (Java и Javascript), лекции будут сопровождаться примерами кода на многих популярных сегодня языках.

# Цель курса

В идеале: каждый, кто успешно закончит курс, будет достаточно квалифицирован, чтобы претендовать на должность Junior Developer в одной из украинских компаний.

Альтернативный вариант: после окончания курса лекций из студентов будет сформирована рабочая группа для участия в совместном проекте.
